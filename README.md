## Sprint 2 Project Idea

For my project I am building a Nursing Shift Manager in Java. The goal is to make a command-line program that helps organize patients and care tasks during a nursing shift.

I have been an LPN for 10 years, and I am interested in working on better EMR systems in the future. I wanted to pick a project that connects with my experience but is still small enough to build for this class.

The program should let a user add patients, add care tasks, record basic vital signs, view pending tasks, update task statuses, and print an end-of-shift summary.

For the final submission, I want to show a working menu-based program that demonstrates a small nursing workflow. I plan to use object-oriented design and add design patterns as the project grows.

I am not trying to build a full electronic medical record system. The main focus is a smaller workflow simulator for tracking nursing responsibilities during one shift.
